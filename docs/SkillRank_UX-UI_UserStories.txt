  1. TONG QUAN HE THONG
================================================================================

1.1 Mo ta du an
---------------
SkillRank la nen tang ket noi sinh vien IT voi doanh nghiep thong qua cac bai
tap (task) thuc te thay vi CV truyen thong. Doanh nghiep dang bai tap, sinh vien
nop bai giai, doanh nghiep cham diem va tim ung vien phu hop.

1.2 Cac vai tro nguoi dung
---------------------------
+----------+----------------------------+------------------------------------------------+
| Vai tro  | Mo ta                      | Quyen han chinh                                |
+----------+----------------------------+------------------------------------------------+
| Student  | Sinh vien IT tim viec      | Duyet task, nop bai, xem ket qua, quan ly ho so|
| Company  | Doanh nghiep tuyen dung    | Tao task, cham bai, xem bang xep hang, quan ly goi|
| Admin    | Quan tri he thong          | Quan ly user, duyet thanh toan, xem thong ke   |
+----------+----------------------------+------------------------------------------------+

1.3 Goi dang ky (Subscription Plans)
-------------------------------------
+----------+----------+------------+-----------------------------------------------+
| Goi      | Gia      | Task/thang | Tinh nang                                     |
+----------+----------+------------+-----------------------------------------------+
| Basic    | Mien phi | 2          | Xem bai nop, cham diem                        |
| Standard | $49.99   | 5          | + Bang xep hang theo task                     |
| Premium  | $99.99   | Khong gioi | + Bang xep hang toan cau, xuat CSV            |
|          |          | han        |                                               |
+----------+----------+------------+-----------------------------------------------+

  3. USER STORIES

--------------------------------------------------------------------------------
3.1 Epic: Xac thuc & Dang ky (Authentication)
--------------------------------------------------------------------------------

US-AUTH-01 | Priority: MUST
  User Story: La sinh vien, toi muon DANG KY TAI KHOAN voi thong tin ca nhan
              va CV de tham gia nen tang.
  Acceptance Criteria:
    - Form dang ky yeu cau: ho ten, email, mat khau, truong, nganh,
      nam hoc, ky nang, SDT
    - Upload CV (PDF, max 5MB)
    - Kiem tra email trung lap
    - Mat khau toi thieu 6 ky tu

US-AUTH-02 | Priority: MUST
  User Story: La doanh nghiep, toi muon DANG KY TAI KHOAN voi thong tin
              cong ty de dang bai tuyen dung.
  Acceptance Criteria:
    - Form dang ky: ten cong ty, ma dang ky, email, mat khau, nganh,
      quy mo, website, dia chi, mo ta
    - Upload giay phep kinh doanh + logo (max 10MB)
    - Kiem tra email va ma dang ky trung

US-AUTH-03 | Priority: MUST
  User Story: La nguoi dung, toi muon DANG NHAP bang email va mat khau
              de truy cap he thong.
  Acceptance Criteria:
    - Chon vai tro (Student/Company) truoc khi dang nhap
    - Token JWT het han sau 7 ngay
    - Redirect dung dashboard theo vai tro

US-AUTH-04 | Priority: MUST
  User Story: La nguoi dung, toi muon DANG XUAT de bao mat tai khoan.
  Acceptance Criteria:
    - Xoa token khoi localStorage
    - Redirect ve trang chu

US-AUTH-05 | Priority: MUST
  User Story: La admin, toi muon DANG NHAP RIENG qua trang admin
              de quan tri he thong.
  Acceptance Criteria:
    - Trang login rieng tai /admin-huong/login
    - Username + password
    - Token rieng (admin_token)

--------------------------------------------------------------------------------
3.2 Epic: Sinh vien - Duyet & Nop bai (Student - Tasks & Submissions)
--------------------------------------------------------------------------------

US-STU-01 | Priority: MUST
  User Story: La sinh vien, toi muon XEM DASHBOARD de nam tong quan hoat dong.
  Acceptance Criteria:
    - Hien thi: tong bai nop, bai dang cho, bai duoc chap nhan
    - Danh sach bai nop gan day

US-STU-02 | Priority: MUST
  User Story: La sinh vien, toi muon DUYET DANH SACH TASK voi bo loc
              de tim bai tap phu hop.
  Acceptance Criteria:
    - Loc theo: nganh, do kho (Easy/Medium/Hard), tu khoa
    - Phan trang (12 task/trang)
    - Hien thi: ten task, cong ty, deadline, do kho, nganh

US-STU-03 | Priority: MUST
  User Story: La sinh vien, toi muon XEM CHI TIET TASK de hieu yeu cau
              truoc khi nop bai.
  Acceptance Criteria:
    - Hien thi: mo ta day du, ket qua mong doi, thoi gian uoc tinh,
      deadline, so luot nop con lai
    - Nut "Nop bai" noi bat

US-STU-04 | Priority: MUST
  User Story: La sinh vien, toi muon NOP BAI GIAI cho task de duoc danh gia.
  Acceptance Criteria:
    - Upload file (PDF/ZIP, max 20MB)
    - Nhap cau tra loi text
    - Nhap link portfolio
    - Moi task chi nop 1 lan
    - Khong nop sau deadline

US-STU-05 | Priority: MUST
  User Story: La sinh vien, toi muon XEM DANH SACH BAI DA NOP va trang thai cham.
  Acceptance Criteria:
    - Hien thi: ten task, ngay nop, trang thai
      (pending/reviewed/interested/rejected)
    - Xem diem va feedback khi da cham

US-STU-06 | Priority: SHOULD
  User Story: La sinh vien, toi muon QUAN LY HO SO CA NHAN de cap nhat thong tin.
  Acceptance Criteria:
    - Chinh sua: ky nang, SDT, link portfolio, LinkedIn
    - Upload lai CV moi
    - Xem avatar

--------------------------------------------------------------------------------
3.3 Epic: Doanh nghiep - Quan ly Task & Danh gia (Company)
--------------------------------------------------------------------------------

US-COM-01 | Priority: MUST
  User Story: La doanh nghiep, toi muon XEM DASHBOARD de theo doi tuyen dung.
  Acceptance Criteria:
    - Hien thi: tong task, tong bai nop, ung vien duoc chap nhan
    - Quota task su dung (theo goi dang ky)

US-COM-02 | Priority: MUST
  User Story: La doanh nghiep, toi muon TAO TASK MOI de danh gia ung vien.
  Acceptance Criteria:
    - Nhap: tieu de, mo ta, ket qua mong doi, thoi gian uoc tinh,
      deadline, do kho, nganh, so luot nop toi da
    - Kiem tra quota task theo goi

US-COM-03 | Priority: MUST
  User Story: La doanh nghiep, toi muon XEM DANH SACH BAI NOP tu tat ca task.
  Acceptance Criteria:
    - Hien thi: ten sinh vien, task, ngay nop, trang thai
    - Loc theo task
    - Download file bai nop

US-COM-04 | Priority: MUST
  User Story: La doanh nghiep, toi muon CHAM DIEM VA DANH GIA bai nop
              cua sinh vien.
  Acceptance Criteria:
    - Cho diem 0-100
    - Viet feedback chi tiet
    - Danh trang thai: reviewed/interested/rejected
    - Sinh vien nhan thong bao sau khi cham

US-COM-05 | Priority: SHOULD
  User Story: La doanh nghiep (goi Standard+), toi muon XEM BANG XEP HANG
              THEO TASK de so sanh ung vien.
  Acceptance Criteria:
    - Xep hang sinh vien theo diem cho tung task
    - Chi kha dung tu goi Standard tro len

US-COM-06 | Priority: COULD
  User Story: La doanh nghiep (goi Premium), toi muon XEM BANG XEP HANG
              TOAN CAU de tim talent.
  Acceptance Criteria:
    - Xep hang toan bo sinh vien tren nen tang
    - Loc theo: truong, nam hoc, ky nang, khoang diem
    - Xuat CSV

US-COM-07 | Priority: MUST
  User Story: La doanh nghiep, toi muon QUAN LY GOI DANG KY
              de mo rong tinh nang.
  Acceptance Criteria:
    - Xem goi hien tai va gioi han
    - Yeu cau nang cap (gui cho admin duyet)
    - Xem lich su yeu cau

US-COM-08 | Priority: SHOULD
  User Story: La doanh nghiep, toi muon QUAN LY HO SO CONG TY.
  Acceptance Criteria:
    - Chinh sua: mo ta, website, dia chi, quy mo
    - Upload logo moi

--------------------------------------------------------------------------------
3.4 Epic: Admin - Quan tri he thong (Admin Management)
--------------------------------------------------------------------------------

US-ADM-01 | Priority: MUST
  User Story: La admin, toi muon XEM DASHBOARD TONG QUAN
              de nam tinh hinh he thong.
  Acceptance Criteria:
    - Thong ke: tong sinh vien, doanh nghiep, task, bai nop
    - So yeu cau pending
    - Doanh thu thang
    - Bieu do phan bo goi dang ky
    - Hoat dong gan day

US-ADM-02 | Priority: MUST
  User Story: La admin, toi muon QUAN LY DANH SACH DOANH NGHIEP.
  Acceptance Criteria:
    - Xem danh sach co phan trang (15/trang)
    - Tim kiem theo ten, email
    - Xem thong tin: goi dang ky, task da dung
    - Xoa doanh nghiep

US-ADM-03 | Priority: MUST
  User Story: La admin, toi muon QUAN LY DANH SACH SINH VIEN.
  Acceptance Criteria:
    - Xem danh sach co phan trang (15/trang)
    - Tim kiem theo ten, email
    - Xem thong tin: truong, diem TB, so bai nop
    - Xoa sinh vien

US-ADM-04 | Priority: MUST
  User Story: La admin, toi muon DUYET YEU CAU NANG CAP GOI
              tu doanh nghiep.
  Acceptance Criteria:
    - Xem danh sach yeu cau voi bo loc trang thai
    - Phe duyet / Tu choi voi ghi chu
    - Tu dong cap nhat goi khi phe duyet

--------------------------------------------------------------------------------
3.5 Epic: Trang cong khai (Public Pages)
--------------------------------------------------------------------------------

US-PUB-01 | Priority: MUST
  User Story: La khach, toi muon XEM TRANG CHU de hieu ve SkillRank.
  Acceptance Criteria:
    - Hero section gioi thieu
    - Tinh nang noi bat
    - Testimonials
    - CTA dang ky

US-PUB-02 | Priority: SHOULD
  User Story: La khach, toi muon XEM TRANG BANG GIA de chon goi phu hop.
  Acceptance Criteria:
    - So sanh 3 goi: Basic, Standard, Premium
    - Hien thi tinh nang tung goi
    - Nut dang ky

US-PUB-03 | Priority: SHOULD
  User Story: La khach, toi muon DUYET TASK CONG KHAI
              de xem co hoi truoc khi dang ky.
  Acceptance Criteria:
    - Xem danh sach task khong can dang nhap
    - Yeu cau dang nhap de nop bai

  4. USER FLOW DIAGRAMS
================================================================================

4.1 Flow: Sinh vien dang ky & nop bai
--------------------------------------

  [Trang chu]  -->  [Dang ky SV]  -->  [Dang nhap]  -->  [Dashboard SV]
  (Landing)        (Form + CV)       (Email+Pass)            |
                                                              |
                                                              v
  [Xem ket qua]  <--  [Nop bai]  <--  [Chi tiet]  <--  [Duyet danh
   & Feedback]       (File+Text+       Task             sach Task]
                      Link)                              (Loc + Tim)


4.2 Flow: Doanh nghiep tao task & cham bai
-------------------------------------------

  [Trang chu]  -->  [Dang ky DN]  -->  [Dang nhap]  -->  [Dashboard DN]
  (Landing)        (Form+License)    (Email+Pass)            |
                                                              |
                   [Cham diem]  <--  [Xem bai nop]  <--------+
                   (0-100 +          (Applicants)             |
                    Feedback)                                 |
                                                              |
                   [Bang xep hang]  <--  [Tao Task]  <--------+
                   (Standard+)           (Form)


4.3 Flow: Nang cap goi dang ky
-------------------------------

  [Company      -->  [Trang         -->  [Gui yeu cau  -->  [Admin nhan
   Dashboard]        Subscription]        nang cap]          thong bao]
                                                                 |
                   [Company       <--  [Admin duyet   <----------+
                    nhan ket qua]       hoac tu choi]
